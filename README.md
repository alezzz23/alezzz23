## <img src="https://user-images.githubusercontent.com/74038190/212284094-e50ceae2-de86-4dd6-9a1c-4eb1b2c6bb35.gif" width="30"> Proyectos Destacados

### 🧠 [NeuroTeach - Plataforma Educativa con IA Emocional]
- **Descripción**: Plataforma innovadora que utiliza affective computing y modelos de lenguaje avanzados para adaptar el contenido educativo según el estado emocional del usuario
- **Tecnologías**: NestJS, React, PostgreSQL, Python, TensorFlow, OpenAI, DeepFace
- **Características**: Detección de emociones vía webcam, tutor IA adaptativo, análisis de patrones de aprendizaje, dashboard de progreso
- 🔗 [Ver proyecto](https://github.com/alezzz23/NeuroTeach)
- ⚡ **Destacado**: Integración de affective computing con educación personalizada

### 🌐 [NetWatch - Monitor de Red Avanzado]
- **Descripción**: Monitor de red en tiempo real con interfaz de terminal (TUI) desarrollado en Go para análisis completo de tráfico y dispositivos
- **Tecnologías**: Go, Docker, TUI, ASCII Graphics
- **Características**: Dashboard en tiempo real, detección automática de dispositivos, análisis de latencia, sistema de alertas
- 🔗 [Ver proyecto](https://github.com/alezzz23/NetWatch)
- 🐳 **Disponible en Docker** con configuración completa

### 🎫 [SGERM - Sistema de Gestión de Eventos y Reservas]
- **Descripción**: Sistema completo para la gestión de eventos, reservas y venta de entradas desarrollado con Spring Boot
- **Tecnologías**: Spring Boot, Java, PostgreSQL, REST API
- **Características**: Gestión de eventos, sistema de reservas, venta de entradas, panel administrativo
- 🔗 [Ver proyecto](https://github.com/alezzz23/-Sistema-de-Gestion-de-Eventos-y-Reservas-SGERM-)
- 🎯 **Enfoque**: Arquitectura empresarial escalable

### 🚀 [Landing Jornada 2025]
- **Descripción**: Landing page moderna y responsiva para evento tecnológico
- **Tecnologías**: HTML5, CSS3, JavaScript, Responsive Design
- **Características**: Diseño moderno, animaciones, optimización móvil
- 🔗 [Ver proyecto](https://github.com/alezzz23/landing-jornada-2025)
- 🎨 **Destacado**: Diseño UI/UX profesional
